# Step 6: Dashboard visualizes metrics in near real-time

## Purpose
The dashboard displays aggregated metrics to end-users in a user-friendly interface.

## Responsibilities
- Fetch metrics from PostgreSQL (and Redis if real-time)
- Display charts and tables
- Provide time-based filtering (day/week/month)
- Show basic funnels and event tracking

## Technology
- Next.js (React framework)
- Chart libraries: Recharts, Visx, or ECharts
- Optional SSR/SSG for performance
